¿Qué es JavaScript?


JavaScript es un lenguaje de programación interpretado, orientado a objetos y basado en eventos.
Es uno de los lenguajes más populares y ampliamente utilizados en el desarrollo web.


Permite a los desarrolladores crear interactividad y dinamismo en las páginas web, lo que mejora la experiencia del usuario. Se utiliza tanto en el lado del cliente (navegador) como en el lado del servidor.


Se define embebido en el documento HTML o como un archivo externos con la extensión .js. Se ejecuta en el navegador del usuario, lo que permite la creación de aplicaciones web interactivas y dinámicas.


Usos y acciones de JS


JavaScript se utiliza para una variedad de tareas, incluyendo:


  • Manipulación del DOM: Permite modificar el contenido y la estructura de una página web.
  • Validación de formularios: Se utiliza para validar datos ingresados por el usuario antes de enviarlos al servidor.
  • Animaciones: Permite crear animaciones y efectos vi
  • Desarrollo de aplicaciones web: Se utiliza en el desarrollo de aplicaciones web modernas, como aplicasuales en las páginas web.
  • Interacción con APIs: JavaScript puede comunicarse con servicios web y APIs para obtener datos dinámicamente.
    • ciones de una sola página (SPA).
  • Desarrollo del lado del servidor: Con Node.js, JavaScript se puede utilizar para crear aplicaciones del lado del servidor.

En resumen, JavaScript es un lenguaje versátil que permite a los desarrolladores crear experiencias web interactivas y dinámicas.

Definición de variables


JS no es un lenguaje con tipado fuerte. Pero tenemos tres tipos diferentes de declarar variables


  • let: Se utiliza para declarar variables que pueden cambiar su valor.
  • const: Se utiliza para declarar variables cuyo valor no puede cambiar una vez inicializado. Es constante.
  • var: Se utiliza para declarar variables, exactamente igual que el let, a diferencia de este var permite acceder a la variable aunque estemos fuera del bloque donde fue declarada, pero debe estar dentro de la misma funcion (Para saber más, mirar Scope y mirar el ejemplo).
